# Hi — I'm Zixi "Oz" Li 👋

> Exploring the structure of reasoning, geometric incompleteness, and computational boundaries — blending mathematics, theory, and code.

I'm a researcher and developer working at the intersection of mathematical reasoning, geometry, and computation. My work combines formal/structural thinking (papers in LaTeX) with reproducible code (Python notebooks and tools) to investigate how reasoning systems behave, collapse, and sometimes reveal surprising connections to classical problems.

- Bridge theoretical ideas (number theory, geometry) with computational exploration.

## Recent Projects

### LLM Infrastructure & Optimization
- **[March](https://github.com/lizixi-0x2F/March)** — High-performance KV cache sharing library. Uses Trie structure to achieve 80-97% memory savings through prefix sharing in LLM inference scenarios. C/Python implementation with zero-copy query access. Reduces memory footprint in multi-turn conversations and batch inference.
  
- **[Collins](https://github.com/lizixi-0x2F/Collins)** — Hash-Adam optimizer. Memory-efficient Adam variant compressing optimizer states (m, v) via 2-Universal Hashing + Count-Sketch, reducing per-parameter state from O(N) to O(K). Fits hash tables in GPU L1/L2 cache.

### Educational/Research
- **[Reasoning Kingdom](https://github.com/datawhalechina/reasoning-kingdom)** — Open-source handbook on AI reasoning mechanisms. A thought experiment manual examining how modern AI systems reason, through information theory, symbolic logic, and representation learning. 13 comprehensive chapters covering from thermodynamic principles to Transformers to computational boundaries. *Co-authored with Datawhale community*.
